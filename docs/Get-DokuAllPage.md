# Get-DokuAllPage

## SYNOPSIS
Get all pages

## SYNTAX

```
Get-DokuAllPage [<CommonParameters>]
```

## DESCRIPTION
Get all pages from the currently connected API endpoint

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-DokuAllPage | Format-Table -AutoSize
```

Gets all pages and displays them in a table
