# Get-DokuPageList

## SYNOPSIS
Gets an array of all pages from an instance of DokuWiki.

## SYNTAX

```
Get-DokuPageList [<CommonParameters>]
```

## DESCRIPTION
Gets an array of all pages from an instance of DokuWiki.

## EXAMPLES

### EXAMPLE 1
```powershell
$AllPages = Get-DokuPageList
```
