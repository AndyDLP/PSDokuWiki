# Get-DokuServerRpcVersionSupported

## SYNOPSIS
Returns the servers minimum supported RPC API version.

## SYNTAX

```
Get-DokuServerRpcVersionSupported [<CommonParameters>]
```

## DESCRIPTION
Returns the servers minimum supported RPC API version.

## EXAMPLES

### EXAMPLE 1
```powershell
$RPCVersionsSupported = Get-DokuServerRpcVersionSupported
```
